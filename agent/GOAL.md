# /agent/GOAL.md

## BloomSpace Goal

Build BloomSpace v1 for iOS App Store.

BloomSpace is a calm productivity app that helps users turn a messy brain dump into a small, doable daily focus plan.

## MVP Goal

The MVP must prove one thing:

> Can the user feel less overwhelmed by turning many tasks into a small Today Focus list?

## Current Scope

Build only:

1. Authentication
2. Brain Dump task inbox
3. Task CRUD
4. Energy selector
5. Today’s Focus recommendations
6. Basic dashboard
7. Settings/logout

## Do Not Build Yet

* AI features
* Calendar sync
* Payments
* Teams
* Social features
* Push notifications
* Habit tracking
* Apple Watch
* Complex gamification

## Success Criteria

The app is successful when:

* User can sign up
* User can log in
* User can add tasks
* User can organize tasks by priority and energy
* User can select energy level
* App suggests 2–4 tasks
* User can complete tasks
* Dashboard updates
* App is stable enough for TestFlight
